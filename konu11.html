Hayırlı olsun hocam. Windows kurmadan önce kesinlikle şu ayarları yapın:

  • D.O.C.P / XMP: Bunu "Profile 1" yapın. RAM'leriniz ancak o zaman 3600 MHz çalışır.
  • Re-Size BAR: "Enabled" yapın (Ekran kartı performansını artırır).
  • Above 4G Decoding: "Enabled" yapın.
  • CSM (Compatibility Support Module): "Disabled" yapın.

Bu ayarları yapıp F10 ile kaydedip çıkın, sonra Windows kurun.

Arkadaşın dediklerine ek olarak, eğer Windows 11 kuracaksanız Secure Boot (Güvenli Önyükleme) ayarını "Windows UEFI Mode" olarak seçtiğinizden emin olun.

Ayrıca AMD fTPM ayarının da "Enabled" (Açık) olması gerekiyor, yoksa Windows 11 kurulumunda hata alırsınız.
